# IPARA — Référence rapide

> Version condensée pour consultation rapide.

---

## Les 5 catégories

**I — Inbox**
Tout ce qui arrive et n'est pas encore trié. Durée de vie max : 7 jours.

**P — Projects**
Objectif défini + ligne d'arrivée. Sous-dossier par projet avec structure standardisée.

**A — Areas**
Responsabilités sans fin : Reconstruction, Relations, etc. Pas de date de fin.

**R — Resources**
Références permanentes : templates, SOPs, prompts, règles système.

**A — Archives**
Tout ce qui est terminé. Immuable.

---

## Décision rapide : où va ce fichier ?

```
C'est temporaire et non trié → Inbox/
C'est lié à un projet actif → Projects/[Nom-Projet]/Notes/
C'est une responsabilité continue → Areas/[Nom-Area]/
C'est une référence réutilisable → Resources/
C'est terminé → Archives/
```
