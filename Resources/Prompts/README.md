# Bibliothèque de prompts SCC

**But :** Centraliser les prompts réutilisables pour opérer, analyser et améliorer le SCC.

Ces prompts sont des outils de travail — pas des prompts de projets (ceux-là restent dans `Projects/*/Prompts/`).

---

## Prompts disponibles

| Fichier | Usage |
|---------|-------|
| `Analyse-Projet-4-Etapes.md` | Analyse complète d'un projet en 4 étapes avec évaluation multi-agents |

---

## Comment ajouter un prompt

1. Créer un fichier `[Nom-Descriptif].md` dans ce dossier
2. Inclure en en-tête : but du prompt, quand l'utiliser, quand ne pas l'utiliser
3. Ajouter une ligne dans le tableau ci-dessus
4. Commit N1

---

## Convention de nommage

`[Sujet]-[Action].md` en kebab-case, en français.

Exemples : `Analyse-Projet-4-Etapes.md`, `Audit-Coherence-SCC.md`, `Creation-Workflow.md`
