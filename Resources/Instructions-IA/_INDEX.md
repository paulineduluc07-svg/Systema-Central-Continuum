# Instructions IA — Index

> Chaque outil IA qui touche à la vie de Paw a son fichier ici.
> Ce dossier est le contrat de travail entre Paw et ses outils.

---

## Règle universelle — Mémoire et apprentissage

**Valable pour TOUS les outils IA sans exception.**

À chaque interaction, si Paw :
- corrige une façon de faire
- précise une préférence
- montre une logique qu'elle utilise
- signale que quelque chose ne lui convient pas

→ L'outil note cette information dans son fichier ici (`## Ce que j'ai appris`) ET dans son propre système de mémoire si applicable.

L'objectif : ne jamais recommencer à zéro. Chaque session doit être plus alignée que la précédente.

---

## Règle universelle — Note de session

À la fin de chaque session de travail, ajouter une note dans le fichier du projet ou de l'area concerné.

Format :
```
[AAAA-MM-JJ] [Nom de l'outil]
Tâche : Ce qui a été fait.
```
