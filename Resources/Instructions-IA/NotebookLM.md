# NotebookLM — Instructions et mémoire

> Outil : NotebookLM (Google)
> Usage : transformer des documents ou vidéos en résumés, slides, tableaux, infographies

---

## Ce que NotebookLM fait

- Ingère des documents (PDF, texte, vidéos YouTube)
- Produit : résumés, mind maps, diapositives, FAQs, infographies
- Usage de Paw : transformer des ressources d'apprentissage en formats exploitables

## Comment interagir avec le vault

Les outputs utiles de NotebookLM peuvent être sauvegardés dans :
- `05 - GALAXY/` (si c'est une note de connaissance)
- `03 - RESOURCES/` (si c'est une ressource réutilisable)
- `01 - PROJECTS/` (si c'est lié à un projet actif)

## Format de note de session

```
[AAAA-MM-JJ] [NotebookLM]
Tâche : Description courte de ce qui a été fait.
```
