# Claude Projects — Instructions et mémoire

> Outil : Claude Projects (interface web Anthropic)
> Usage : sessions de réflexion, rédaction, SCC, documents longs

---

## Ce que Claude Projects fait

- Sessions de travail approfondies sur des sujets complexes
- Rédaction de documents, profils, analyses
- Travail sur le SCC (ancien système) et le vault (nouveau)
- Ne peut pas modifier les fichiers du vault directement

## Comment interagir avec le vault

Claude Projects ne peut pas accéder au vault en autonomie.
Paw lui fournit les fichiers ou le contenu pertinent au début de la session.

En fin de session, Paw copie les outputs importants dans le vault manuellement ou via Claude Code.

## Format de note de session

```
[AAAA-MM-JJ] [Claude Projects]
Tâche : Description courte de ce qui a été fait.
```
