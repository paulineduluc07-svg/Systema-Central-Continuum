# SOP — SCC : Constitution et règles

> Le Systema Central Continuum est le second cerveau numérique de Paw.
> Source de vérité unique pour toutes les interactions avec les outils IA.
> Version 1.0 — Mars 2026

---

## C'est quoi

Chaque document dans ce vault représente un contexte de vie réel — identité, projets, relations, travail personnel. Ces documents sont vivants : ils grandissent avec le temps, mais ne se dupliquent jamais.

---

## Règle 1 — Un seul endroit par information

Chaque information existe à un seul endroit dans le vault.
Jamais copiée, jamais dupliquée dans un autre fichier.
Si une info existe déjà quelque part, on y fait référence — on ne la réécrit pas.

## Règle 2 — Les IA ajoutent, elles ne modifient pas

Quand un outil IA travaille dans un document, il ne modifie pas le contenu existant.
Il ajoute uniquement une **note de session** à la fin du document.

## Règle 3 — Chaque note de session est datée et identifiée

Format obligatoire :

```
[2026-MM-JJ] [Plateforme]
Tâche : Ce qui a été fait.
```

Plateformes : `[Claude Code Terminal]`, `[Claude Projects]`, `[Claude Cowork]`, `[ChatGPT]`, `[Gemini]`, `[Grok]`, etc.

---

## Ce que le SCC n'est pas

- Un dossier séparé dans le vault (c'était l'ancienne structure — migrée)
- Un système parallèle à Obsidian
- Un endroit pour les fichiers de code
