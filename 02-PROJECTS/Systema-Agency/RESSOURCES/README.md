# RESSOURCES/

Documentation technique et matériel de travail du projet Systema-Agency.

- `AGENT_PLANS/` — plans d'implémentation rédigés pour/par les agents IA
  (un fichier par chantier, daté `AAAA-MM-JJ-sujet.md`). Les plans terminés
  vont dans `AGENT_PLANS/done/`.

Règle : ce dossier = matériel de référence, pas de notes perso (elles vont
dans `NOTES_DE_PAULINE.md`) ni de secrets (ils vont dans `Assets/SECRETS.md`,
jamais commité).
