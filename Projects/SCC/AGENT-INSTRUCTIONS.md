# AGENT-INSTRUCTIONS.md — SCC (Systema Central Continuum)

Ce projet gère la construction et l'évolution du SCC lui-même.
Le SCC est traité comme un projet à part entière, avec ses propres tâches, roadmap et mémoire opérationnelle.

## Contexte du projet
- **Objectif principal :** Finaliser la gouvernance du SCC et le faire approuver par Paw avant d'alimenter les projets actifs.
- **État actuel :** Phase 1 — Construction (governance, workflows, règles en cours de finalisation)
- **Stack tech :** Aucune — repo documentaire Markdown + GitHub
- **Priorité :** Priorité absolue. Rien d'autre ne démarre avant que le SCC soit approuvé par Paw.

## Règles spécifiques à ce projet

1. Toute modification des règles fondamentales (AGENTS.md, Multi-Agent-Collaboration.md) est **N4** — instruction explicite de Paw uniquement.
2. Les workflows peuvent être complétés ou clarifiés en **N2** — avec trace.
3. Ce projet n'a pas de `Code/` actif — c'est un projet documentaire.
4. Les décisions sur la structure SCC doivent toutes être tracées (type `decision`).

## Où noter les traces de ce projet

- Décisions SCC → `Resources/Systeme/Multi-Agent-Collaboration.md` (trace transversale)
- Notes opérationnelles → `Projects/SCC/Notes/`
