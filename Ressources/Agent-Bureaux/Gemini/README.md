# Bureau Gemini

Ce dossier est le bureau operateur de Gemini dans le SCC.

## Role
- Centraliser les preferences de collaboration specifiques a Gemini
- Clarifier comment Paw veut parler a Gemini et ce que Gemini doit surveiller
- Eviter de disperser ces preferences dans plusieurs fichiers

## Garde-fous
- Ce bureau n'override jamais AGENTS.md
- Ce bureau ne remplace jamais les traces projet
- Ce bureau ne sert pas a stocker le travail metier des projets
- Toute execution reelle et toute trace de travail doivent rester dans les fichiers du projet concerne
- Ce bureau doit rester minimal, stable et a forte valeur

## Contenu
- PROFILE.md -- personnalisation de travail Gemini
- REQUEST-PATTERNS.md -- facons de formuler les demandes a Gemini
- WATCHPOINTS.md -- points de vigilance permanents pour Gemini
