# Bureau Codex

Ce dossier est le bureau operateur de Codex dans le SCC.

## Role
- Centraliser les preferences de collaboration specifiques a Codex
- Clarifier comment Paw veut parler a Codex et ce que Codex doit surveiller
- Eviter de disperser ces preferences dans plusieurs fichiers

## Garde-fous
- Ce bureau n'outrepasse jamais `AGENTS.md`
- Ce bureau ne remplace jamais les traces projet
- Ce bureau ne sert pas a stocker le travail metier des projets
- Toute execution reelle et toute trace de travail doivent rester dans les fichiers du projet concerne
- Ce bureau doit rester minimal, stable et a forte valeur

## Contenu
- `PROFILE.md` -- personnalisation de travail Codex
- `REQUEST-PATTERNS.md` -- facons de formuler les demandes a Codex
- `WATCHPOINTS.md` -- points de vigilance permanents pour Codex
